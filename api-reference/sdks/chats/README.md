### Example Usage

```python
import os
from syllable_sdk import Syllable

s = Syllable(
    api_key_header=os.getenv("SYLLABLE_API_KEY_HEADER", ""),
)

res = s.agents.chats.send_message(request={
    "service_name": "<value>",
    "source_name": "<value>",
    "chat_id": "<id>",
    "agent_id": "<id>",
    "org_name": "<value>",
    "text": "<value>",
})

if res is not None:
    # handle response
    pass

```
